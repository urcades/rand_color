# Security Policy

## Supported versions

This project is in maintenance mode.

- Latest published versions of workspace crates are supported for security fixes.
- Older versions may receive fixes at maintainer discretion, but no guarantee is provided.

## Reporting a vulnerability

Please report vulnerabilities privately by opening a private security advisory on GitHub:

- Repository: https://github.com/urcades/rand_color
- Security advisories: https://github.com/urcades/rand_color/security/advisories

If you cannot use GitHub advisories, contact the maintainer directly.

## Response expectations

- Initial triage target: within 7 days
- Status update target: within 14 days
- Fix/release timing depends on severity and reproducibility

## Disclosure

Please avoid public disclosure until a fix is available and users have had a reasonable upgrade window.
